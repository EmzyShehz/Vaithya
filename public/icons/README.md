# PWA Icons

This folder should contain your app icons in the following sizes:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## How to Generate Icons

You can use tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- https://favicon.io/

Upload your Vaithya logo (ideally a square SVG or high-res PNG) and download all sizes.

## Icon Guidelines

- Use your green (#337e51) brand color
- Make sure the icon works on both light and dark backgrounds
- Consider using a "maskable" design (safe area in the center)
- The icon should be recognizable at small sizes
